# Calculadora Moderna Responsive

Una **calculadora web moderna**, elegante y totalmente **responsive**, creada con **HTML, CSS y JavaScript**.
Funciona perfectamente en **celulares  y laptops**, con animaciones.

## Características principales

Operaciones básicas: suma, resta, multiplicación y división
Dos pantallas:
- Operación arriba
- Resultado abajo
Historial de operaciones
- Se puede mostrar / ocultar
- Click para reutilizar una operación
 Botón **DEL** para borrar un solo dígito
 Soporte para:
- Teclado físico
- Teclado numérico
- Mouse

##  Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
